# AboutMe

* This is my first React Native app.

## Getting Started

* Download and unzip the project.

## Running app

```sh
    cd Lab_01_About_Me

    npm install (to install dependencies)

    react-native run-android (to run on Android emulator)
    react-native run-ios (to run on iOs emulator)

```
